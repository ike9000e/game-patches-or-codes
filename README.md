# Game Patches Or Codes
Collection of created cheat codes or memory addresses for video games. Currently included are Pnach paches for GT4 game on PCSX2. Converted, ported or created patches. Author: ike9000.

## Covered games:
* Gran Turismo 4 Online Public Beta, SCUS-97436 (U). 
  Short name used: GT4OL.

## How To Use

Copy .pnach files to the "/cheats" sub-directory under the PCSX2 installation.

Enable cheats by opening PCSX2 and going in Settings->Emulation, and check the Enable Cheats checkbox.

Do note that each game has its own "enable cheats" option. Right click on a game in the game list and navigate to Proprieties->Cheats.

To reload the contents of the pnach file when the game is running select Tools->Reload Cheats/Patches from the toolbar.
